# Sistema de Gestao de loja em C

## 3. Funcionalidades do Sistema

 ### 3.1 Gestão de Produtos
Cadastro de produtos (ID, nome, preço, estoque)

Listagem de produtos

Atualização de estoque

Remoção de produtos

Alerta de estoque baixo

Busca de produtos por nome ou ID

Alerta de validade 


### 3.2 Sistema de Vendas
Registro de vendas

Seleção de múltiplos produtos

Definição de quantidade

Cálculo automático do valor total

Atualização automática do estoque

 ###  3.3 Controle de Caixa
Registro do total vendido no dia

Histórico de movimentações

Abertura e fechamento de caixa

Consulta de saldo atual

  ### 3.4 Gestão de Funcionários
Cadastro de funcionários (ID, nome, cargo, senha)

Listagem de funcionários

Controle de cargos (ADMIN / FUNCIONÁRIO)

  ### 3.5 Sistema de Login
Login individual com ID e senha

Controle de acesso baseado no cargo: 
  ADMIN: acesso total
  FUNCIONÁRIO: acesso restrito (vendas)

  ### 3.6 Comissão de Funcionários
Cálculo de comissão por venda

Percentual configurável

Registro do total vendido por funcionário

  ### 3.7 Metas e Desempenho
Definição de metas de vendas

Acompanhamento de desempenho individual

Comparação entre meta e resultado

  ### 3.8 Relatórios
Total de vendas por dia

Produtos mais vendidos

Faturamento geral

Lucro estimado

Desempenho por funcionário

Produtos mais vendidos
